# Bash Scripting Suite for System Maintenance

A set of easy-to-use Bash scripts to help you take care of your Linux system. This collection can help with things like backups, cleaning up files, checking system health, and keeping an eye on logs.

## It can:

- **Backups**: Quickly save important files and folders.
- **Log Monitoring**: Watch your log files and get alerts for issues.
- **System Cleanup**: Remove unnecessary files to free up space.
- **Health Checks**: See how your disk, memory, and services are doing.
- **Customizable**: You can tweak scripts to fit your needs.

## Requirements

- Linux or Unix-like OS  
- Bash shell  
- Some scripts might need `sudo` to run
